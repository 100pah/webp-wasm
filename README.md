# webp-wasm

## Install
```shell
sh build.sh install
```

## Build
```shell
sh build.sh
```

## Release
```shell
sh build.sh release
```

## Code style

+ C: based on https://google.github.io/styleguide/cppguide.html (with 4 space indent)
+ JS: based on https://google.github.io/styleguide/jsguide.html (with 4 space indent)
